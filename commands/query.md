# /query

Wiki'ye soru sor. `agents/researcher/skills/WIKI_QUERY.md` talimatlarını uygula.

## Kullanım

```
/query Hook tekniklerinin engagement üzerindeki etkisi nedir?
```

## Adımlar

1. obsidian-hybrid-search MCP `search(soru)` çağır → ranked snippets
2. İlgili sayfaları oku
3. Cevabı sentezle — her iddia kaynağa bağlı
4. Cevap yeni sentez içeriyorsa `wiki/syntheses/` altına geri-dosyala
5. `wiki/log.md`'ye kayıt yaz

## Wiki'de Cevap Yoksa

"Bu sorunun cevabı wiki'de yok. Şu kaynaklar ingest edilirse yanıtlanabilir: [öneri]" de.
Tahmin veya kaynaksız iddia üretme.
